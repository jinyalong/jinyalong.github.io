HTML5

什么是HTML

  • HTML 指的是超文本标记语言: HyperText Markup Language

  • HTML 不是一种编程语言，而是一种标记语言

  • 标记语言是一套标记标签 (markup tag)

  • HTML 使用标记标签来描述网页

  • HTML 文档包含了HTML 标签及文本内容

  • HTML文档也叫做 web 页面

W3C标准

  • W3C

    • World Wide Web Consortium（万维网联盟）

    • 成立于1994年，Web技术领域最权威和具有影响力的国际中立性技术标准机构

    • http://www.w3.org/

    • http://www.chinaw3.org/

  • W3C包括

    • 结构化标准语言（HTML、XML）
    • 表现标准语言（CSS）
    • 行为标准语言（DMO、ECMAScript）

网页基本标签

图像标签

src和alt必填。

链接标签

  • href标签：必填，表示链接地址。

  • target标签：表示窗口在哪里打开

    • _bank：在新标签打开
    • _self：在自己的网页中打开

页面链接：页面之间的跳转

锚链接

  • 定义一个标记点
  • 通过#标记名跳转

功能性链接

例如：mailto

行内元素和块元素

  • 块元素

    • 无论多少内容，该元素独占一行
    • （p、h1-h6…)

  • 行内元素

    • 内容撑开的元素，左右都是行内元素的可以排在一行

列表

分类：

  • 无序列表
  • 有序列表
  • 自定义列表

表格标签

  • table：表格标签
  • tr：行标签
  • td：列标签
  • tr colspan=“3”：跨列
  • td rowspan=”2“：跨行

媒体元素

视频和音频

视频元素：video

  • src：资源路径
  • controls：控制栏
  • autoplay：自动播放

音频元素：audio

  • src：资源路径
  • controls：控制栏
  • autoplay：自动播放
